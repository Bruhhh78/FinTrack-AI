# FinTrack AI

FinTrack AI is a modern, full-stack personal finance management application built with the MERN stack (MongoDB, Express, React, Node.js). It empowers users to take complete control of their financial health through intuitive tracking, intelligent insights, and beautiful visualizations.

## 🌟 What This Project Does

- **Expense & Income Tracking:** Seamlessly log multiple income streams and daily expenses with automatic categorization and tagging.
- **Dynamic Budgeting:** Set monthly budgets per category and track your spending in real-time. Receive visual indicators when approaching limits.
- **Financial Goals:** Create custom savings goals, track progress, and monitor deadlines to ensure you hit your targets.
- **Comprehensive Analytics:** Gain deep insights into spending patterns with interactive pie charts, bar graphs, and 12-month trend visualizations powered by Recharts.
- **Financial Health Scoring:** Get a calculated wellness metric that assesses your overall savings rate, budget adherence, and financial stability.
- **Premium Aesthetics:** Features a modern, responsive "glassmorphism" design with smooth micro-animations, customizable dark/light modes, and a deeply intuitive user experience.

## 🚀 Deployment Architecture

This project is built to run as a **Split Deployment**:
- **Frontend (React/Vite):** Hosted on **Netlify** for blazingly fast global CDN delivery and smooth client-side routing.
- **Backend (Node/Express):** Hosted on **Vercel** as serverless functions, ensuring instant scalability for the API endpoints.
